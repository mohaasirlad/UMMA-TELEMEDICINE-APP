# UMMA Telemedicine App

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Platform](https://img.shields.io/badge/platform-Android-green.svg)

UMMA Telemedicine is a production-ready Android application designed to bridge the gap between patients and healthcare providers. It features a robust real-time messaging system, video consultation capabilities, and a dual-portal architecture.

## 🚀 Key Features

### For Patients
- **Doctor Discovery**: Search and filter doctors by specialty and rating.
- **Appointment Booking**: Secure scheduling with integrated payment simulation.
- **Symptom Checker**: Preliminary health self-assessment tool.
- **Telehealth**: High-quality video consultations via Twilio.

### For Doctors
- **Management Dashboard**: Real-time overview of appointments and patient requests.
- **Patient Records**: Secure access to consultation history.
- **Earnings Tracking**: Integrated revenue reporting.

### Communication
- **Real-time Chat**: Persistent messaging using Room + Firebase.
- **Push Notifications**: Instant updates via FCM.

## 🛠️ Technical Stack
- **Language**: Java
- **Database**: Room (Local), Firebase Realtime Database (Cloud)
- **Networking**: OkHttp, Twilio Video SDK
- **Architecture**: MVVM with LiveData
- **UI**: Material Design 3
